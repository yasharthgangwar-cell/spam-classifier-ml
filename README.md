Spam Email Classifier using Machine Learning
About the Project

This project is a machine learning-based spam detection system that classifies messages as spam or ham (not spam). It uses natural language processing techniques to process text data and train a classification model.

Objective

The objective of this project is to build a model that can automatically detect spam messages using machine learning and NLP techniques.

Dataset

The dataset used is the SMS Spam Collection dataset, which contains labeled text messages:

Ham: Normal messages
Spam: Unwanted or promotional messages

Model Used
Logistic Regression / Naive Bayes
Performance
Accuracy: ~97%
Evaluation metrics: Precision, Recall, F1-score
Strong performance on spam detection task
